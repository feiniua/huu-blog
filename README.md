# huu-blog
个人博客，采用vue + springboot前后端分离实现，springboot只提供数据接口。

## 前端

### vue

**使用依赖**

* router

* axios

* element-ui

* mavon-editor

* marked

* js-md5

**项目结构**

* src
  * components/   // 自定义组件
  * config/       // 后端提供url信息
  * router/       // 路由信息
  * views/        // 页面
  * App.vue       
  * main.js

## 后端

采用Springboot。

* shiro:
	* 继承FormAuthenticationFilter拦截器，对不同请求方式进行拦截
* querydsl：
	* DAO层用来多表查询

### 建表

Article：用来记录文章信息

Tag：标签，用来对文章进行分类。

*认证*

User：用户，一个用户对应一个角色

Role：角色，一个角色对应多个权限

Permission：权限，add对应post，edit对应put，delete对应delete
